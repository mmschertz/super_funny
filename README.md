# SuperFunny

Welcome to Super Funny. Jokes included are SFW and not religious, political, racist or sexist in nature.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'super_funny'

```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install super_funny

## Usage

$ super_funny
